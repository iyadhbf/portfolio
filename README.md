# 💼 Iyadh Belfetni - Developer Portfolio

This is my personal portfolio website built using **HTML**, **TailwindCSS**, and a **Node.js + Express** backend with **Nodemailer** integration to handle contact form submissions via email.

🟢 **Live Site**: [https://iyadhbelfetni.onrender.com](https://iyadhbelfetni.onrender.com)

---

## 🚀 Features

- 🔹 Clean, responsive front-end built with **TailwindCSS**
- 🔹 Contact form with **email sending functionality** using **Nodemailer**
- 🔹 Backend developed with **Express.js**
- 🔹 Deployed for free on **Render**
- 🔹 Modular and easy-to-update codebase

---

## 🛠️ Technologies Used

- **Frontend**: `HTML`, `TailwindCSS`
- **Backend**: `Node.js`, `Express.js`
- **Email**: `Nodemailer`
- **Deployment**: `Render`

---

### 🧪 Local Installation & Setup

To run this project locally, follow these steps:

#### 1. 📥 Install Dependencies

```bash
npm install
```


### 2. 🔐 Create a .env File

In the root directory, create a file named `.env` with the following content:

```ini
EMAIL_USER=your-gmail-address@gmail.com
EMAIL_PASS=your-app-password
PORT=3000
```

### 3. ▶️ Run the Server

```bash
npm start
```
